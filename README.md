### 运行环境

支持WordPress版本：5.6+

兼容Chrome、Firefox、Safari等主流浏览器

支持设备：响应式布局，不同设备不同展示效果

服务器环境建议：PHP 7.2+和MySQL 5.6+

### 主题特点

 1. 和酱茄Free小程序开源版源码配套；

 2. 同步酱茄Free小程序开源版浏览/评论/用户头像等信息；

 3. 首页幻灯片可分别设置图片和链接；

 4. 可隐藏指定分类目录；

 5. 列表/文章元素支持（头像/作者/面包屑/浏览/版权信息/猜你喜欢等）开启/停用；

 6. 免装SEO插件，文章关键词/描述自动获取；

 7. 响应式设计并兼容pc和移动端；

 8. 内置标签聚合/友情链接等单页；

 9. 内置热门文章/标签云/猜你喜欢/链接等小工具；

 10. 文章自动获取正文第一张图为缩略图；

 11. 文章列表无限加载；

 12. 文章内容结尾可选显示版权说明；
 
 13. 文章支持打赏、点赞、评论。
